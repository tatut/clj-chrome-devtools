request-entries

(request-entries)(request-entries {:as params, :keys [cache-id skip-count page-size]})(request-entries connection {:as params, :keys [cache-id skip-count page-size]})

Requests data from cache.

Parameters map keys:

Key Description
:cache-id ID of cache to get entries from.

:page-size Number of records to fetch.

Return map keys:

Key Description
:cache-data-entries Array of object store data entries.
